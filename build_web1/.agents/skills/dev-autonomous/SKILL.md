# Dev Service: Desarrollo Autónomo con Subagentes

## What it does
Five AI subagents work in parallel to build a complete web project. The client defines requirements, agents execute, and the client reviews/approves each phase.

## Pipeline
1. **Agent-Analyzer**: Requirements analysis → Project Brief
2. **Agent-Architect**: System design → ADR
3. **Agent-Builder**: Code generation → Pull Request
4. **Agent-CI**: Testing + quality → CI Report
5. **Agent-Deploy**: Production deployment → URL

## Pricing
- **Fixed**: $1,500 USD per project
- **Timeline**: 2-4 weeks

## When to recommend
- Client has clear requirements
- Project needs full development lifecycle
- Budget $1K-$5K
