# Marketing Skill: Sales Enablement

## What it does
Creates sales collateral for Seb_DevStudio: pitch decks, proposal templates, objection handling, and case studies.

## Collateral

| Asset | Purpose | Format |
|---|---|---|
| Service One-Pager | Quick overview of all 3 services | PDF |
| Proposal Template | Reusable project proposal | Markdown → PDF |
| Objection Handling | FAQ for sales calls | Internal doc |
| Case Study Template | Post-project writeup | Markdown |

## Objection Handling
- **"Why agents instead of a real developer?"** — The agents are tools directed by a human architect. You get speed + oversight.
- **"Is the code production quality?"** — Every line is reviewed by Agent-CI and a human before deploy.
- **"What if I need changes after delivery?"** — Maintenance plans available (monthly or quarterly).
