# Dev Service: Consultoría Técnica

## What it does
Code audits, architecture definition, CI/CD optimization, and legacy-to-modern migration plans.

## Deliverables
- **Code Audit Report**: Findings, risks, recommendations
- **Architecture Decision Record**: Stack + structure decisions
- **Migration Plan**: Step-by-step with timeline
- **Executive Summary**: Non-technical overview for stakeholders

## Pricing
- **Fixed**: $2,500 USD per engagement
- **Timeline**: 1-3 months

## When to recommend
- Client has existing codebase
- Architecture is unclear or outdated
- Budget over $2,500
- Need for knowledge transfer to internal team
