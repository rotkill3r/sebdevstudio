# Marketing Skill: Social Demos

## What it does
Short-form video and screenshot demos showing the agent pipeline in action.

## Content Types
- **Agent walkthrough**: Screen recording of 5 agents executing
- **Time-lapse build**: Fast-forward from blank page to deployed site
- **Side-by-side**: Before/after with agent output visible
- **Client reaction**: Testimonial paired with agent dashboard

## Platforms
- LinkedIn: Technical audience, longer form
- Twitter/X: Short clips, dev community
- YouTube Shorts: High-level overviews

## Frequency
- 2-3 posts per week during active projects
- 1 case study per completed project
