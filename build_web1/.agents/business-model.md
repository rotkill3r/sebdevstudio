# Seb_DevStudio — Business Model

## Revenue Streams

| Service | Price | Target Client | Volume Potential |
|---|---|---|---|
| Desarrollo Autónomo | $1,500/project | Founders, SMBs | High |
| Web + AI Tools | $800/project | Early-stage startups | Very High |
| Consultoría Técnica | $2,500/project | Tech teams, agencies | Medium |

## Cost Structure
- **Fixed**: Domain ($12/yr), Vercel Pro ($20/mo), Formspree (free tier)
- **Variable**: AI tool API usage, third-party services per project
- **Labor**: Time invested per project (varies by complexity)

## Go-to-Market
- Landing page (this project)
- Social demos on LinkedIn/Twitter showing agent pipeline
- Cold outreach to local businesses needing web presence
- Referral program with previous clients
